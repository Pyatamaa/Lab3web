# lab3_php_koneksi
repository khusus untuk penempatan tugas pertemuan 3 menggunakan bahasa PHP

---

## Penjelasan
Program Create, Read, Update, dan Delete(CRUD) sederhana dengan menggunakan Bahasa Program PHP dan koneksi ke Database. Data yang sudah diinput akan ditampilkan dengan Array.

![screenshot](https://user-images.githubusercontent.com/92738041/230710614-29097282-b387-4547-bdd7-8f857b495830.png)

Proses Create menggunakan metode Post dapat diakses dengan klik button berwarna hijau dengan tulisan "Tambah Barang". Selanjutnya, tinggal input data sesuai keinginan dan pilih kategori sesuai keinginan.

![Screenshot (96)](https://user-images.githubusercontent.com/92738041/230710527-778b946e-c7c4-4908-a2d9-b8185bccd714.png)

Untuk mengubah data dapat mengakses tombol "Ubah" berwarna kuning, lalu akan muncul pilihan untuk mengubah data.

![Screenshot (98)](https://user-images.githubusercontent.com/92738041/230710798-2a938db3-a7e4-491b-80b6-3a462f5a1d91.png)

Lalu untuk yang terakhir untuk menghapus data yang sudah diinput bisa dengan klik tombol "Hapus" dengan warna merah, ketika diklik akan muncul message untuk memastikan apakah ingin benar-benar menghapus data data yang dipilih atau tidak.

![Screenshot (99)](https://user-images.githubusercontent.com/92738041/230710949-75049e63-f534-40f5-bb83-32be29eb13ce.png)
